QUICK START
--------------------------------------------------------------------------------

1. Access the WebUI:
   Open http://localhost:8888 in your browser

2. Create an API Token:
   Use the WebUI to create an authentication token

3. Import a Cloud Image:
   Navigate to Images → Import
   Example: Ubuntu Cloud Image URL
   
4. Create a Network:
   Navigate to Networks → Create
   
5. Create a Storage Pool:
   Navigate to Storage → Create
   
6. Create a VM:
   Navigate to VMs → Create
   Select your image, network, and storage pool

7. Start the VM:
   Click Start on the VM detail page
   
8. Access Console:
   Click Console tab to access VM terminal

--------------------------------------------------------------------------------
MANAGEMENT COMMANDS
--------------------------------------------------------------------------------

Stop services:
  sudo pkill -f chv-controller
  sudo pkill -f chv-agent

Start services:
  sudo /srv/data02/projects/chv/start-chv.sh

Or using systemd:
  sudo systemctl start chv-agent
  sudo systemctl start chv-controller

View logs:
  sudo tail -f /var/lib/chv/logs/controller.log
  sudo tail -f /var/lib/chv/logs/agent.log
