Install CHV

CHV is a Linux-first, cloud-image-first virtualization platform for sovereign private cloud and edge environments.

Run the all-in-one installer on a fresh Ubuntu server with root access:

curl -sfL https://get.cellhv.com/ | sh -

Or pin a specific version:

curl -sfL https://get.cellhv.com/ | INSTALL_CHV_VERSION=0.0.0.2 sh -

Read the full deployment guide for manual setup, build instructions, and multi-node expansion.
